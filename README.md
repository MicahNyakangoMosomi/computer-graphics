## Design Overview:
The house is divided into three main zones:
### Public Zone
Entrance, Living Room
### Semi-Public Zone
Dining Room, Kitchen,
### Private Zone
Master Bedroom, Bedroom 1, Bedroom 2, Bathrooms
## Entrance & Living Room
The main entrance opens directly into the living room
The living room acts as a central gathering space
It provides access to:
Bedrooms,Extra room,Bathroom
This placement prevents visitors from passing through private areas
